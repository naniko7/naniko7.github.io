1、统计学习

1.1 统计学习简述

  • 基于数据构建模型，使用模型预测数据。
  • 统计学习也成为统计机器学习。
  • 研究对象：数据。获取数据的特征，抽取数据的模型，回归数据的分析与预测。
  • 方法
    • 监督学习
    • 无监督学习
    • 强化学习
  • 三要素
    • 模型：模型的假设空间（某个函数的集合）
    • 策略：模型的评价准则，用于选出最优模型
    • 算法：选出最优模型

1.2 统计学习分类

1.2.1 基本分类

监督学习
  • 本质：学习输入到输出的映射的统计规律
  • 基础知识
    • 三个空间
      • 输入空间
      • 输出空间：可以看作特征空间
      • 特征空间：具体的输入是一个实例，由特征向量表示，特征向量存在的空间称为特征空间
    • 模型假设：输入数据与输出数据遵循联合概率分布。训练数据与测试数据被看作都是依照联合概率分布独立同分布产生的。
    • 假设空间：输入空间到输出空间的映射的集合
    • 可以是概率模型或者非概率模型
      • P（y|x）
      • y = f(x)
    • 可用场景
      • 回归
      • 分类
      • 标注
无监督学习
  • 描述：从无标注数据中学习预测模型的机器学习问题。
  • 训练模型：X是输入空间，Z是隐式结构空间
    • z = g(x)
    • P(x|z)
    • P(z|x)
  • 可用场景
    • 聚类
    • 降维
    • 概率估计
强化学习

  • 本质：学习最优的序贯决策

  • 过程：

    • 系统从环境中观察到一个状态s和一个奖励r
    • 系统根据s和r采取一个动作a
    • 环境根据系统选择的动作决定下一步的状态和下一步的奖励

  • 相关参数：

    • 五元组<S,A,P,r,y>——用于确定状态和奖励
      • S：状态。下一个状态只依赖于前一个状态和动作。
      • A：动作。
      • P：状态转移概率函数：由动作和状态预测下一个状态。
      • r：奖励函数。下一个奖励只依赖前一个状态和动作。
      • y：衰减系数
    • 策略Π——用于确定动作
      • 给定状态下动作的函数 a = f(s)或者 P（a|s）。给定策略后，出现一个状态就能确定一个动作，即确定了系统与环境的互动行为。
    • 价值函数/状态价值函数
      • 定义：策略Π从某一个状态s开始的长期累积奖励的数学期望
      • 关键：
        • 条件：状态s = st
        • 奖励从rt+1开始计算
    • 动作价值函数
      • 定义：策略Π从某一个状态s和动作a开始长期累积奖励的数学期望
      • 关键：
        • 条件：状态s = st ，动作a = at

  • 学习目标：

    • 在所有可能的策略中选出价值函数最大的策略Π^*^

  • 方法：

    • 无模型方法：不直接学习模型
      • 基于策略：试图求解最优策略
      • 基于价值：试图求解最优价值函数，间接学到最优策略
    • 有模型方法
      • 直接学习马尔可夫决策过程的模型：状态转移概率函数P与奖励函数r

【Notes】

  • 学习中往往从具体的策略出发，不断优化已有策略。
半监督学习与主动学习

半监督学习：

  • 少量标注数据和大量未标注数据

主动学习：

  • 机器给出示例让教师标注，根据标注数据学习预测模型
  • 目标：找出对学习最有帮助的示例让教师标注，减小标注的代价。

【Notes】

  • 监督学习是基于已经给定的标注数据，是随机的，而不是特定的选择的标注数据，可以看作是”被动学习“

1.2.2 按模型分类

概率模型与非概率模型

监督学习

  • 概率模型：P（y|x）

    • 属于生成模型

  • 非概率模型：y = f(x)

    • 属于判别模型

无监督学习

  • 概率模型：P（z|x）或者P（x|z）
  • 非概率模型：z = g(x)

概率模型举例：决策树、朴素贝叶斯、隐马尔可夫模型、条件随机场、概率潜在语义分析、潜在狄利克雷分配、高斯混合模型

非概率模型举例：感知机、支持向量机、k近邻、AdaBoost、k均值、潜在语义分析、神经网络

【Notes】

  1. P(y|x)和y = f(x)可以相互转化。条件概率分布最大化后得到函数，函数归一化后得到条件概率分布。
  2. 概率模型和非概率模型区别不在于映射关系，在于模型的内在结构。
    • 概率模型一定可以表示为联合概率分布的形式，非概率模型不一定存在这样的联合概率分布
  3. 概率模型代表：概率图模型。联合概率分布可以根据图的结构分解为因子乘积形式。
线性模型与非线性模型
  • 非概率模型可以分为线性模型和非线性模型。如果y = f(x)是线性函数，就是线性模型。

线性模型举例：感知机、线性支持向量机、k近邻、k均值、潜在语义分析

非线性模型举例：核函数支持向量机、AdaBoost、神经网络

参数化模型与非参数化模型
  • 参数化模型：模型参数的维度固定
  • 非参数化模型：参数维度不固定或者无穷大，随着训练数据量增多而增大

1.2.3 按算法分类

在线学习
  • 每次接受一个样本，进行预测，学习模型，不断重复操作
  • 既可以是无监督学习也可以是监督学习，强化学习本身就是一种在线学习
  • 通常模型准确率低于批量学习，因为每次可以利用的数据有限。优点在于可以快速适应新的模式（此时就不满足独立同分布假设了）
批量学习
  • 一些接受所有数据，学习模型，进行预测

1.3 统计学习方法三要素

方法 = 模型 + 策略 + 算法

模型

  • 模型：所要学习的条件概率分布或者决策函数
    • 假设空间里包含所有可能的条件概率分布或者决策函数
    • 假设空间为决策函数的集合

策略

一些名词概念
  • 损失函数
    • 用于度量模型一次预测的好坏，用于度量预测错误的程度，是f(x)和Y的非负实值函数，L（Y，f(X)）
    • 常用损失函数
      • 0-1损失函数
      • 平方损失函数
      • 绝对损失函数
      • 对数损失函数
  • 风险函数（期望损失）
    • 损失函数的期望
    • 又要最小化期望损失，联合分布概率又未知，又要学习联合分布概率——>监督学习变成病态问题
  • 经验风险（经验损失）
    • 模型关于训练样本集的平均损失。只要样本足够多，就可以用经验风险去估计期望损失。
经验风险最小化与结构风险最小化
  • 什么是经验风险最小化？有什么作用？有什么注意事项？
    • 经验风险最小化是指是的经验风险或损失达到最小以此保证获得最优模型。
    • 作用：当样本足够大时，可以保证很好的学习效果。
      • 极大似然估计就是经验风险最小化的例子。
    • 注意：当样本容量小时，会产生”过拟合“现象。
  • 什么是结构风险最小化？有什么作用？
    • 为了防止过拟合提出来的策略。
    • 结构风险最小化等价于正则化。
    • 结构风险在经验风险上加上表示模型复杂度的正则化项或者罚项。

【Notes】

  1. 泛函：任意的向量空间到标量的映射。也就是从函数空间到数域的映射。

1.4 模型评估与模型选择

概念

  1. 训练误差与测试误差
    • 测试误差具有更好的预测能力
  2. 过拟合与模型选择
    • 过拟合：学习选择的模型所包含参数过多，已知数据上预测效果好，但是未知数据上预测很差。

1.5 正则化与交叉验证

正则化

  • 作用：选择经验风险和结构风险同时较小的模型。保证预测准确的同时模型最简单。

交叉验证

  • 简单交叉验证
  • S折交叉验证
  • 留一交叉验证

1.6 泛化能力

  • 泛化能力：对未知数据的预测能力
  • 泛化误差上限
    • 含义
    • 证明

1.7 生成模型与判别模型

生成模型：学习联合概率分布P（X,Y）

优势：

  • 可以还原出联合概率分布
  • 收敛速度更快
  • 存在隐变量时，仍然可以用生成方法学习

判别模型：直接学习决策函数或者条件概率分布

优势：

  • 学习的准确率更高
  • 方便对数据进行各种程度上的抽象、定义特征并使用特征，可以简化学习问题

1.8 监督学习应用

1.8.1 分类问题

  • 分类问题包括学习和分类两个过程。

    • X可以是离散的也可以是连续的
    • Y取有限个离散值

  • 分类性能指标：分类准确率。

  • 二分类问题常用评价指标：

    • 精确率P

    • 召回率R

    • F1 score： 2/F = 1/P + 1/R

      【Notes】：当精确率和召回率都高时，F1也会高

1.8.2 标注问题

  • 标注问题是分类问题的推广。
    • 输入：一个观测序列
    • 输出：一个标记序列或者状态序列
    • 过程：学习和标注两个过程

1.8.3 回归问题

  • 回归问题的学习等价于函数拟合
  • 常用损失函数：平方损失函数
    • 使用平方损失函数时，可以用最小二乘法求解
